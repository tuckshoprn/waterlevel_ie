## Submitting to Home Assistant Brands

To have these logos appear in the Home Assistant frontend, they need to be submitted to the official Home Assistant brands repository:

### Submission Process

1. Fork the repository: https://github.com/home-assistant/brands
2. Create a new directory: `custom_integrations/waterlevel_ie/`
3. Add the four logo files (icon.png, icon@2x.png, logo.png, logo@2x.png)
4. Create a pull request with:
   - Clear title: "Add WaterLevel.ie custom integration branding"
   - Description explaining the integration
   - Link back to this repository

### Requirements Checklist

- [x] Icon is 256x256px (1:1 aspect ratio)
- [x] Icon@2x is 512x512px
- [x] Logo is landscape oriented
- [x] Logo@2x is high-resolution
- [x] All files are PNG with transparent backgrounds
- [x] All files are optimized and interlaced
- [x] No Home Assistant branding used
- [x] Files trimmed of excess whitespace

### After Submission

Once the pull request is merged into the Home Assistant brands repository:
- Logos will automatically appear in Home Assistant frontend
- Users will see branding in integrations list
- HACS will display the logos properly
- May take up to 24 hours for Home Assistant to pick up new branding
